Tentang file arsip terkompresi

Arsip terkompresi menggabungkan beberapa file ke dalam satu file sehingga memudahkan pengangkutan atau penyimpanan di ruang disk. Perangkat lunak pengarsipan juga dapat menyediakan opsi untuk enkripsi, pembekuan, checksum, ekstraksi diri, dan pemasangan sendiri. Zip adalah format yang paling banyak digunakan, yang digunakan oleh sistem operasi Windows dan baru-baru ini oleh OSX juga. RAR juga merupakan format yang sangat populer dan fleksibel. Unix menggunakan format file tar, sedangkan Linux menggunakan format tar dan gz.
